Research Projects

Industrial Defect Detection

Developed an AI-driven defect detection system for industrial applications, leveraging deep learning techniques to identify and classify defects in manufacturing processes with high accuracy.

Image Fusion for Intelligent Transportation

Designed and implemented a multisensor image fusion framework for enhanced perception in intelligent transportation systems, integrating fusion, detection, and segmentation techniques to improve object recognition and scene understanding.

Multimodal stress classification

Investigating and developing deep learning-based multimodal techniques combining textual data, facial expressions, and voice data for accurate and early stress detection. Implemented advanced neural network architectures to improve classification performance and robustness.

UAV Sensor‑Failure Classification and Diagnostics

Designed fault‑classification models using multi‑sensor signals to detect and categorize sensor anomalies during UAV operations. Evaluated performance on simulated and real flight data to support future health‑monitoring and fault‑tolerant control strategies.

Adaptive Power‑Electronics Control with Machine Learning

This project involved the development of a machine learning-based adaptive power-electronics controller for real-time control of power systems under changing load, voltage, and operating conditions. The project aims to maintain output stability and power quality across operating ranges. This project is currently ongoing and in the initial stages.
